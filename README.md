Анализ рынка недвижимости РФ за 2018–2021 годы и построение модели предсказания цены за м².

## Данные
https://drive.google.com/file/d/1c56OodTuY8DwyeILIdtAr_aUVmA_gZAD/view?usp=sharing

- **realty_18_21.csv** — ~5.5 млн объявлений о недвижимости (цена, площадь, комнаты, этаж, регион и др.)
- **region.csv** — справочник регионов (код → название)
- **key_rate.csv** — ключевая ставка ЦБ РФ по датам

## Что сделано

### Предобработка
- Удалено 235 655 дубликатов
- Восстановлено 261 677 записей с отрицательным числом комнат (подтверждена гипотеза об ошибке знака через RandomForest, accuracy 90.6%)
- Отфильтрованы аномалии: этаж > этажности дома, кухня > общей площади
- Выбросы удалены по 1-му и 99-му перцентилям log(цена/м²) в разрезе регионов (~104 884 записи)

### Анализ рынка
- Устойчивый рост цен с 2018 по 2021 год по всем регионам
- Самые дорогие регионы: Москва, Санкт-Петербург, Сахалинская область
- Максимальный рост цен за период: Амурская область (+81%), Ямало-Ненецкий АО (+86%)
- Сезонность: дешевле покупать летом (июнь–июль), дороже зимой (ноябрь–февраль)
- Корреляция ключевой ставки ЦБ с ценой: −0.20 (ставка падает → цены растут)

### Моделирование

**Random Forest** — использовался для классификации комнат и оценки важности признаков. Главный фактор цены — регион (~67%), далее этажность дома (~10%) и ключевая ставка (~6%).

**Предсказательная модель:
- R² = 0.80
- RMSE = 24 514
- MAE = 15 302
- MAPE = 22.81%
- Переобучения нет (разница train/test RMSE ~4 единицы)

Модель хорошо работает для массового сегмента, но занижает цены дорогих объектов (300 000+ руб./м²).
